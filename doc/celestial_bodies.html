class CelestialBody(builtins.object)
    CelestialBody(name, mass, orbital_radius, simulated_radius, type_of_object, central_mass, vRelative, colour)
 
Class used to represent the different bodies present in the solar system,
this includes planets, stars and satellites.

get_initial_velocity(self, position, velocity)
Method used to get the correct tangential velocity of an object orbiting.
This is calculated by taking the perpendicular vector to the position (constained
in the orbit's plane) and multiply it by the modulus (norm) of the velocity.
Args:
    position ([float]): Numpy Array containing the position of the object at time =0,
                        position is relative to  the central star at (0,0).
    velocity (float): modulus of the velocity.
 
Returns:
    [float]: returns the new velocity vector tangent to the orbit's trajectory.

update_acceleration(self, others)
Methods used to update the acceleration experienced by a body.
The acceleration at any time is assumed to be only given by the gravitational
interaction with other bodies (other potential forces are omitted as they
should be negligible on the scale of the solar system).
 
Args:
    others ([CelestialBody]): other Celestial Bodies that influence the total
                       acceleration of the given body.
Raises:
    ValueError if the distance between two bodies is found to be 0.
update_position_beeman(self, time_step)
Method used to update the position  of a Celestial Body using Beeman's method.
The position of the object at time t + dt depends on the position, the velocty and
acceleration at time t and the acceleration at t -dt and the time between updates dt.
This method also checks if a period has been completed, if so it will print to the console.
Args:
    time_step (float): time between updates.
